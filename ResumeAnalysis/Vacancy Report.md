## 13. КЛЮЧЕВЫЕ ВЫВОДЫ

1. **Python — абсолютный стандарт** (76.5% вакансий). Без продвинутого Python вход невозможен.

2. **PyTorch победил TensorFlow** в соотношении 1.9:1 (29% vs 15%). Учи PyTorch первым.

3. **NLP/LLM — самое горячее направление** (30-45% вакансий). Лучшая ниша для входа без диплома — направление молодое, академический background менее критичен.

4. **74.5% вакансий не требуют диплом** явно (на основе 548 вакансий). Подтверждается артефактами (MADE+HH: <10% для ML/DS).

5. **Математика нужна реально** (58.8% вакансий). Линал, теорвер, статистика — спрашивают на собеседовании у всех грейдов.

6. **MLOps стал обязательным** — Docker (~42%), Kubernetes (~27%), Airflow (~20%). DevOps-опыт — преимущество.

7. **Удалёнка доступна в 48%** вакансий (263 из 548). Больше в стартапах и продуктовых компаниях, меньше в банках.

8. **Junior позиций мало** (22.5%), позиций без опыта — 2.2%. Стажировки в крупных компаниях — реальная точка входа.

9. **Зарплата Middle ML: 200-400K**, Senior: 350-600K+, Lead: 500K-950K. Конкуренция для middle — 2-3 резюме/вакансию.

10. **CatBoost — российская специфика** (7.7% вакансий). Встречается чаще, чем на западных рынках. Обязательно знать для собеседований.

11. **scikit-learn жив** (15.3%) — классический ML для табличных данных по-прежнему актуален, не только deep learning.

12. **SQL — второй обязательный навык** (41.4%). Нельзя игнорировать даже ML-инженерам.
