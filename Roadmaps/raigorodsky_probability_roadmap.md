## Блок I: Фундамент оценок (Лекции 01–03)

**Зачем для ML:** Любая модель — это оценка параметров. Понимание свойств оценок (bias, consistency) напрямую транслируется в bias-variance tradeoff и выбор моделей.

### Лекция 02 — Функции риска и штрафа (11.02.2021)

**Суть:** Как формально сравнивать оценки. Функция штрафа → функция риска. Это предок loss-функций в ML.

| Таймкод | Тема |
|---------|------|
| [00:00] | Напоминание: точечные оценки, обозначения |
| [05:00] | Функция штрафа: квадратичная (MSE), абсолютная (MAE) |

### Лекция 06 — Теорема о MLE. Фишер. Рао-Крамера. Доверительные интервалы (11.03.2021)

**Суть:** Почему MLE работает. Нижняя граница для дисперсии любой оценки. Доверительные интервалы.

| Таймкод | Тема |
|---------|------|
| [00:00] | Анонс: теорема о свойствах MLE |
| [05:00] | Условия регулярности (когда MLE "хорошо себя ведёт") |
| [10:00] | Лог-правдоподобие, дифференцирование плотности |
| [15:00] | E[score function] = 0, дисперсия score function |
| [20:00] | Информация Фишера: определение, смысл |
| [25:00] | Неравенство Рао-Крамера: Var(θ̂) ≥ 1/I(θ) |
| [30:00] | MLE: состоятельность + асимптотическая нормальность при регулярности |
| [35:00] | Нерегулярные модели: равномерное распределение |
| [40:00] | Интервальные оценки: уровень доверия |
| [45:00] | Построение CI для нормального: известная дисперсия |
| [50:00] | CI при неизвестной дисперсии: хи-квадрат |

**Связь с ML:** Информация Фишера определяет "кривизну" поверхности likelihood → в ML это Hessian loss-функции → используется в Natural Gradient Descent, Adam приближает. Рао-Крамера говорит: есть теоретический минимум ошибки, ниже которого нельзя опуститься — аналог irreducible error. Доверительные интервалы — основа A/B тестирования и uncertainty estimation.
