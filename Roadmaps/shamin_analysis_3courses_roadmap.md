### Лекция 02: Случайные величины, распределения, E[X], Var, Cov ⭐

| Тема | Статус | Комментарий |
|------|--------|-------------|
| **Случайные величины** | **🟢** | **Формальное определение СВ как функции ω → ℝ. Ты знаешь через практику, но формализм полезен для собесов** |
| **Функция распределения** | **🟢** | **⭐ F(x) = P(X ≤ x). Свойства: монотонность, правонепрерывность. Спрашивают на собесах** |
| **Плотность распределения** | **🟢** | **⭐ f(x) = F'(x). Связь с вероятностью через интеграл. Для непрерывных СВ** |
| **Математическое ожидание** | **🟢** | **E[X] — ты знаешь формулу. Здесь — формальные свойства: линейность, E[g(X)]** |
| **Дисперсия и ковариация** | **🟢** | **⭐ Var = E[(X-μ)²] = E[X²] - (E[X])². Cov(X,Y) — прямая связь с корреляцией и ковариационной матрицей (PCA!)** |
| **Независимые СВ** | **🟢** | **E[XY] = E[X]·E[Y], Var[X+Y] = Var[X] + Var[Y]. Фундамент для понимания batch statistics в BatchNorm** |

**Вердикт: 🟢 СМОТРЕТЬ ЦЕЛИКОМ (~90 мин).** Это ядро теорвера для ML. Всё здесь напрямую используется: E[X] → loss function, Var → нормализация, Cov → PCA/корреляция. Даже если ты знаешь формулы — формальные свойства и доказательства закрепят.

---

### Лекция 03: Чебышёв, ЗБЧ, ЦПТ, Пуассон ⭐

| Тема | Статус | Комментарий |
|------|--------|-------------|
| **Неравенство Чебышёва** | **🟢** | **⭐ P(\|X−μ\| ≥ kσ) ≤ 1/k². Спрашивают на собесах. Основа для ЗБЧ** |
| **Закон больших чисел** | **🟢** | **⭐ Среднее выборки → E[X]. Почему SGD работает: средний градиент по мини-батчу сходится к истинному градиенту** |
| **Центральная предельная теорема** | **🟢** | **⭐⭐ Сумма iid → N(μ,σ²/n). Обоснование: почему нормальное распределение везде, почему confidence intervals работают, почему batch size влияет на стабильность обучения** |
| **Предельная теорема Пуассона** | **🟡** | **Биномиальное → Пуассон при n→∞, p→0. Менее критично для ML, но спрашивают** |

**Вердикт: 🟢 СМОТРЕТЬ ЦЕЛИКОМ (~90 мин).** Чебышёв + ЗБЧ + ЦПТ — тройка, которую спрашивают на каждом втором ML-собесе. У тебя в LearnState: "CLT и LLN — знаю хорошо". Здесь — формальные доказательства + неравенство Чебышёва (которое ты ещё не разбирал подробно).

### Лекция 04: Случайные процессы, Гауссовские, Марковские цепи

| Тема | Статус | Комментарий |
|------|--------|-------------|
| Определение случайного процесса | 🟡 | X(t,ω) — функция времени и случая |
| Классы случайных процессов | 🟡 | Обзорно |
| **Гауссовские процессы** | **🟢** | **⭐ GP — используются в Bayesian optimization (подбор гиперпараметров). Определяется средним + ковариационной функцией. Спрашивают на собесах уровня Senior** |
| Стационарные процессы | 🟡 | Статистические свойства не меняются во времени |
| **Марковские цепи** | **🟢** | **⭐ P(X_{n+1}\|X_n,...,X_1) = P(X_{n+1}\|X_n). Прямая связь: (1) языковые модели = марковское предположение (n-gram), (2) MCMC для сэмплирования, (3) HMM, (4) RL. Спрашивают** |

**Вердикт: 🟢 СМОТРЕТЬ (~90 мин, можно на 1.5x бегло по обзорным частям).** Гауссовские процессы и марковские цепи — два столпа, которые пронизывают ML. Даже если ты идёшь в NLP/LLM — марковское свойство это фундамент, от которого трансформеры отталкиваются (и преодолевают).

### Лекция 05: Мат. статистика, интервальные оценки, линейная регрессия

| Тема | Статус | Комментарий |
|------|--------|-------------|
| **Математическая статистика** | **🟢** | **Обратная задача: по данным → оцениваем параметры. Ты знаешь MLE, здесь — общий фреймворк** |
| **Выборочные моменты** | **🟢** | **⭐ Выборочное среднее, дисперсия, стандартное отклонение. Смещённая vs несмещённая оценка (n vs n−1). Спрашивают** |
| **Интервальные оценки** | **🟢** | **⭐ Confidence intervals. Закроет пробел из LearnState: "Confidence intervals — нужно добить"** |
| **Линейная регрессия** | **🟢** | **⭐ Статистический взгляд на linreg (не ML-шный). OLS = MLE при нормальных ошибках. Глубже понимание** |
| **Коэффициент детерминации R²** | **🟢** | **R² = 1 − SS_res/SS_tot. Метрика качества регрессии. Спрашивают** |

**Вердикт: 🟢 СМОТРЕТЬ ЦЕЛИКОМ (~90 мин).** Каждая тема — прямой вопрос на ML-собесе. Интервальные оценки закроют конкретный пробел из LearnState. Линейная регрессия через статистику — второй взгляд на то, что ты уже реализовал.

## Курс 2: Теория оптимизации (4 лекции) ⭐⭐ САМЫЙ ЦЕННЫЙ КУРС

### Лекция 01: Математическое программирование — теория

| Тема | Статус | Комментарий |
|------|--------|-------------|
| Введение: min f(x), x ∈ X | 🟢 | Формальная постановка задачи оптимизации |
| Инфимум, минимизирующие последовательности | 🟡 | Почему min может не достигаться |
| Метрические пространства, компактность | 🟡 | Ты уже знаешь |
| **⭐ Парето-оптимальность** | **🟢** | **Multi-objective optimization: accuracy vs speed, precision vs recall. Спрашивают** |
| **⭐ Локальные vs глобальные минимумы** | **🟢** | **Почему GD застревает в local min. Необходимые условия ∇f = 0** |
| **⭐ Метод множителей Лагранжа** | **🟢** | **Из перспективы оптимизации (не матанализа). → SVM** |
| **⭐ Выпуклое программирование** | **🟢** | **Convex f + convex X → local = global. Почему linreg/logreg сходятся, а нейросети — нет** |

**Вердикт: 🟢 СМОТРЕТЬ (полностью, ~90 мин, первые 30 мин можно на 1.5x).** Парето + convex optimization + Лагранж = три темы, которые спрашивают на ML-собесах.

---

### Лекция 02: Численные методы оптимизации ⭐⭐ ЛУЧШАЯ ЛЕКЦИЯ ВСЕХ 3 КУРСОВ

| Тема | Статус | Комментарий |
|------|--------|-------------|
| **Зачем численные методы** | **🟢** | **Аналитическое решение ∇f = 0 почти всегда невозможно** |
| **Алгоритмы поиска** | **🟢** | **Обзор подходов** |
| **⭐⭐ Градиентный метод** | **🟢** | **x_{n+1} = x_n − α·∇f(x_n). Learning rate, направление спуска. Это то, что ты делаешь при W -= lr·dW** |
| **⭐ Проблемы GD на практике** | **🟢** | **Zigzagging, выбор lr, медленная сходимость в "оврагах". Прямая мотивация для momentum, Adam. Ответ на собесовый "почему Adam лучше SGD"** |
| Вычисление производных численно | 🟡 | Конечные разности. Для понимания autograd |
| **⭐ Метод штрафов** | **🟢** | **Constrained → unconstrained. ПРЯМАЯ связь: L1/L2 регуляризация = штраф за сложность модели. min(loss) + λ·\|\|w\|\|² это и есть метод штрафов** |
| Проблемы метода штрафов | 🟡 | Нюансы выбора λ |
| Генетические алгоритмы (обзор) | 🔴 | Не для MLE |

**Вердикт: 🟢 СМОТРЕТЬ ЦЕЛИКОМ (~90 мин, последние 5 мин можно пропустить).** Gradient descent + его проблемы + метод штрафов = регуляризация. Закрывает сразу несколько пробелов. Это лекция, после которой L2-регуляризация, momentum, Adam встанут в единую картину.
