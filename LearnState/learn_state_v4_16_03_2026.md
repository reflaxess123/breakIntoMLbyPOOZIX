### 1.2 Теория вероятностей и статистика (~80%)

**✅ Знаю глубоко:**
- **Аксиоматика Колмогорова** — пространство элементарных исходов Ω, событие как подмножество, σ-алгебра (замкнутость относительно счётных объединений), вероятностная мера P, тройка (Ω, 𝒜, P). Зачем нужна σ-алгебра: обычная алгебра не работает с бесконечными объединениями → нельзя определить пределы, непрерывные величины, функцию распределения. Парадокс Банаха-Тарского — почему нельзя определить вероятность на всех подмножествах
- **Операции над множествами** — объединение (OR), пересечение (AND), дополнение (NOT), разность (AND NOT), симметрическая разность (XOR), подмножество (импликация). Законы де Моргана, дистрибутивность, коммутативность. Связь с булевой алгеброй и программированием
- **Комбинаторика** — перестановки (n!), размещения (n!/(n-k)!), сочетания (n!/k!(n-k)!), упорядоченные выборки с возвращением (Mⁿ). Задача о днях рождения: точная формула и приближение через e. Задача о лотерее (гипергеометрическая). Трюк "считай через дополнение"
- **Биномиальное распределение** — P(Aₖ) = Cₙᵏ pᵏ qⁿ⁻ᵏ, смысл каждой части (число раскладов × вероятность одного расклада)
- **Теорема Байеса** — prior × likelihood = posterior, числовые примеры, сценарии с сильным/слабым prior
- **MLE (Maximum Likelihood)** — полный путь: произведение → логарифм → сумма → cross-entropy. Связь MLE→CE, Normal→MSE, Categorical→CE
- **MAP (Maximum A Posteriori)** — вывод L2 = нормальный prior, L1 = лапласовский prior
- **Распределения** — Normal, Bernoulli, Binomial подробно; Poisson, Geometric, Exponential, Student, Beta, Uniform на уровне обзора; категориальное — подробно через softmax; хи-квадрат — на уровне распределения (не тест)
- **Cross-entropy** — полный вывод из MLE, связь с энтропией Шеннона, формула H(P,Q) = -Σ P·ln(Q)
- **Мат.ожидание, дисперсия** — E(X) как взвешенная сумма, Var = E(X²) − E(X)², выборочная оценка
- **CDF, квантили, перцентили** — эмпирическая CDF, Q1/Q2/Q3, боксплот, выбросы через IQR
- **Ковариация и корреляция** — определения, формулы, связь с ковариационной матрицей
- **ЦПТ (Central Limit Theorem)** — средние выборок → нормальное распределение
- **Bootstrap** — ресемплинг с возвращением, построение доверительных интервалов, OOB-концепция

**✅ Знаю на рабочем уровне:**
- **Нормальное распределение** — стандартизация Z = (X−μ)/σ, z-таблица (Φ), правило 68-95-99.7
- **Энтропия Шеннона** — мера неопределённости, формула H = -Σ p·ln(p)
- **Hypothesis testing** — p-value (определение, интерпретация), H₀ vs H₁, ошибка I рода (α), ошибка II рода (β)
- **Доверительные интервалы** — построение через разные распределения (z, t, χ²), связь с ЦПТ, ширина CI ∝ 1/√n
- **Z-тест, t-тест** — одновыборочный, двухвыборочный, когда какой использовать
- **Распределение Стьюдента** — когда n мало и σ неизвестно → t вместо z
- **Вилкоксон** (непараметрический тест) — разобран
- **Bias-Variance Tradeoff** — концептуально понимаю (underfitting = high bias, overfitting = high variance, регуляризация добавляет bias, уменьшает variance), но формальный вывод MSE = Bias² + Var + Noise не делал
- **KL-дивергенция** — обзорно, в связке с cross-entropy: H(P,Q) = H(P) + D_KL(P‖Q), понимание что D_KL измеряет "расстояние" между распределениями

**⚠️ Частично / в процессе:**
- **Хи-квадрат тест** — начал, не дочитал (распределение χ² знаю, сам тест — нет)
- **Распределение Фишера и F-тест** — знаю что существует, не разбирал
- Условная вероятность — через Байеса, но формально совместные распределения и маргинализацию не разбирал

**❌ Не знаю:**
- A/B-тестирование — как применять тесты на практике, sample size, множественное тестирование (Bonferroni)
- Совместные распределения и маргинализация (формально)
- Bias-Variance Tradeoff — формальный вывод MSE = Bias² + Var + Noise

### 1.3 Матанализ и оптимизация (~55%)

**✅ Знаю глубоко:**
- **Частные производные, градиент** — градиент как вектор частных производных, направление максимального роста через cos θ, визуализация для матриц весов
- **Chain rule** — три звена для softmax regression: dL/dP · якобиан softmax (10×10) · dZ/dW = Xᵀ. Понимаю что chain rule = основной клей backprop
- **Backpropagation** — полный вывод для softmax regression: dL/dW = (1/m)·Xᵀ·(P−Y). Якобиан, сокращение, элегантная финальная формула
- **Gradient descent (SGD)** — W -= lr·dW, реализация с нуля, mini-batch, усреднение по батчу

**❌ Не знаю:**
- Adam optimizer (momentum + RMSprop)
- Learning rate schedules (warmup, cosine annealing)
- Convex vs non-convex оптимизация

## Главная сила

Подход через **реализацию с нуля**: не заучиваю формулы, а разбираю код строчка за строчкой и параллельно учу математику. Softmax regression на MNIST дал: матричное умножение, softmax, cross-entropy, MLE, chain rule, backprop, SGD, якобиан — всё через одну модель.

Глубокое понимание фундамента: не просто "знаю формулу Байеса", а понимаю зачем нужна σ-алгебра, как из трёх аксиом вырастает вся теория вероятностей, почему cross-entropy = MLE.

Бэкграунд в VFX (Houdini) даёт интуитивное понимание симуляций, частиц, итеративных процессов — это переносится на gradient descent и оптимизацию.
