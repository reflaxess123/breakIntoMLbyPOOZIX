### Лекция 1: Введение. Метод Монте-Карло. Выборочные характеристики

**Приоритет:** 🟡 Средняя

Николаев вводит философию статистики: модели ошибочны, но полезны. Метод Монте-Карло как способ оценки характеристик, когда аналитическое решение недоступно. Эмпирическое распределение, выборочные характеристики.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Введение | Философия статистики: модели ошибочны, но полезны | 🟡 Полезно | Контекст для понимания ML |
| Монте-Карло | ⭐ **Метод Монте-Карло: оценка характеристик через симуляцию** | 🟢 Смотреть | Используется в MCMC, байесовских методах |
| Выборка | Выборка, эмпирическое распределение, вариационный ряд | 🟡 Полезно | Базовые определения |
| ЗБЧ | Закон больших чисел: почему выборочное среднее сходится | 🟢 Смотреть | Фундамент стохастической оптимизации (SGD) |

**Что вынести:** Метод Монте-Карло — универсальный инструмент для оценки величин, которые нельзя посчитать аналитически. Эмпирическое распределение приближает истинное при росте выборки.

**Пререквизиты:** Теория вероятностей, случайные величины, матожидание, дисперсия.

### Лекция 2: Выборочные характеристики. Базовые свойства оценок ⭐

**Приоритет:** 🟢 Высокая

Центральная лекция по свойствам статистических оценок. Несмещённость, состоятельность, асимптотическая нормальность. Выборочное среднее, дисперсия, квантили.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Выборочные характеристики | Выборочное среднее, дисперсия, медиана, квантили | 🟢 Смотреть | Базовый язык статистики |
| Несмещённость | ⭐ **Смещённость оценки: E[θ*] = θ или нет** | 🟢 Ключевое | «Почему делим на n-1 в выборочной дисперсии?» |
| Состоятельность | ⭐ **Состоятельность: сходимость по вероятности к θ** | 🟢 Ключевое | Фундамент для понимания сходимости в ML |
| Асимпт. нормальность | ⭐ **ЦПТ: sqrt(n)(θ* - θ) → N(0, σ²)** | 🟢 Ключевое | Основа для доверительных интервалов и A/B тестов |
| MSE | Среднеквадратическая ошибка: MSE = bias² + variance | 🟢 Ключевое | Bias-variance tradeoff — топ-вопрос на собесах |

**⭐ Главные инсайты:**
- MSE = bias² + variance — фундаментальная декомпозиция, объясняющая регуляризацию
- Несмещённая оценка не всегда лучшая: иногда смещённая оценка с меньшей дисперсией предпочтительнее (Ridge regression)
- Асимптотическая нормальность — основа для построения доверительных интервалов

**Связь с собесами:** «Что такое bias-variance tradeoff?» — один из самых частых вопросов. Эта лекция даёт строгое обоснование.

**Пререквизиты:** Лекция 1.

### Лекция 3: Дельта-метод. Бутстрап. Интервальные оценки ⭐

**Приоритет:** 🟢 Высокая

Дельта-метод для наследования асимптотической нормальности. Бутстрап как plug-in оценка Монте-Карло. Доверительные интервалы: точные и асимптотические.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Дельта-метод | Если θ* асимпт. нормальна и g дифф., то g(θ*) тоже | 🟡 Полезно | Технический инструмент |
| Бутстрап | ⭐ **Бутстрап: замена истинного распределения на эмпирическое** | 🟢 Ключевое | «Как работает bootstrap?» — спрашивают |
| Доверительные интервалы | ⭐ **CI: покрытие с вероятностью 1-α** | 🟢 Ключевое | Интерпретация CI — частый вопрос |
| Асимпт. интервалы | Интервалы на основе ЦПТ: θ* ± z_{α/2} · se | 🟢 Смотреть | Используются в A/B тестах |
| Бутстрап-интервалы | Перцентильный и BCa бутстрап | 🟡 Полезно | Практический инструмент |

**⭐ Главные инсайты:**
- Бутстрап = «подставь эмпирическое распределение вместо истинного и симулируй»
- Доверительный интервал — это НЕ «вероятность попадания параметра в интервал», а свойство процедуры
- Бутстрап работает когда аналитические формулы недоступны — часто используется для оценки неопределённости в ML

**Связь с собесами:** Bootstrap используется для оценки неопределённости моделей. Доверительные интервалы — основа A/B тестирования.

**Пререквизиты:** Лекции 1-2.

### Лекция 4: Параметрические модели. Оценка параметров ⭐

**Приоритет:** 🟢 **КРИТИЧЕСКАЯ**

Параметрические, непараметрические и полупараметрические модели. Метод максимального правдоподобия (MLE). Метод моментов. Параметрический бутстрап. Достаточные статистики.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Модели | Параметрические vs непараметрические vs полупараметрические | 🟢 Смотреть | Контекст для выбора моделей в ML |
| Парам. бутстрап | Параметрический бутстрап: замена F на F_θ* | 🟡 Полезно | Альтернатива непараметрическому бутстрапу |
| Метод моментов | ⭐ **Метод моментов: приравнять выборочные моменты к теоретическим** | 🟢 Смотреть | Простой метод оценки параметров |
| MLE | ⭐ **Метод максимального правдоподобия: L(θ) = ∏ f(xᵢ|θ) → max** | 🟢 Ключевое | MLE — основа loss-функций в ML |
| Свойства MLE | ⭐ **MLE: состоятельность, асимпт. нормальность, эффективность** | 🟢 Ключевое | Почему MLE — «лучшая» оценка |
| Достаточные стат. | Достаточные статистики, факторизационная теорема | 🟡 Полезно | Теоретическая красота |
| Центр. функция | Метод центральной функции для точных интервалов | ⚪ Теория | Глубокая математика |

**⭐ Главные инсайты:**
- MLE максимизирует вероятность данных при данных параметрах → log-likelihood = сумма логарифмов
- MLE для нормального распределения → MSE loss; MLE для Бернулли → cross-entropy loss
- MLE асимптотически эффективна: достигает нижней границы Рао-Крамера
- Метод моментов — альтернатива MLE, проще но менее эффективен

**Связь с собесами:** «Выведите cross-entropy из MLE» и «Почему MSE для регрессии?» — стандартные вопросы. Эта лекция даёт полный математический фундамент.

**Пререквизиты:** Лекции 1-3.

### Лекция 5: М-оценки. Робастность. Эффективность

**Приоритет:** 🟡 Средняя

Обобщение MLE — M-оценки (минимизация ρ-функции). Робастность к выбросам. Информация Фишера. Неравенство Рао-Крамера (нижняя граница дисперсии). Эффективность оценок. Стабилизация дисперсии.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| М-оценки | ⭐ **M-оценки: обобщение MLE через min Σ ρ(xᵢ, θ)** | 🟡 Полезно | Связь с loss-функциями в ML |
| Робастность | ⭐ **Робастные оценки: медиана vs среднее при выбросах** | 🟢 Смотреть | «Когда MSE плохо? Что делать с выбросами?» |
| Инф. Фишера | ⭐ **Информация Фишера: I(θ) = E[(∂logf/∂θ)²]** | 🟢 Смотреть | Фундамент для понимания сложности задачи оценивания |
| Рао-Крамер | Неравенство Рао-Крамера: Var(θ*) ≥ 1/(n·I(θ)) | 🟡 Полезно | Теоретический предел точности |
| Эффективность | Асимптотическая эффективность MLE | ⚪ Теория | Почему MLE «оптимален» |
| Стабилизация | Стабилизация дисперсии (variance-stabilizing transform) | 🟡 Полезно | Технический трюк |

**⭐ Главные инсайты:**
- M-оценки — общий класс: MLE = частный случай с ρ = -log f
- Медиана робастна к выбросам, среднее — нет. Huber loss — компромисс
- Информация Фишера показывает «количество информации» о параметре в одном наблюдении
- Чем больше информация Фишера, тем точнее можно оценить параметр

**Связь с собесами:** Робастность и Huber loss спрашивают при обсуждении регрессии. Информация Фишера — для продвинутых позиций.

**Пререквизиты:** Лекция 4 (MLE).

### Лекция 6: Проверка гипотез: основы ⭐

**Приоритет:** 🟢 **КРИТИЧЕСКАЯ**

Нулевая и альтернативная гипотезы. Статистика критерия. Ошибки I и II рода. p-value. Мощность критерия. Множественные сравнения.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Формализация | ⭐ **H₀ vs H₁, статистика критерия, критическая область** | 🟢 Ключевое | Базовый язык A/B тестирования |
| Ошибки | ⭐ **Ошибка I рода (α) = ложное отклонение H₀; Ошибка II рода (β) = пропуск эффекта** | 🟢 Ключевое | «Что такое ошибки I и II рода?» — стандарт |
| p-value | ⭐ **p-value: вероятность получить такие или более экстремальные данные при H₀** | 🟢 Ключевое | «Что такое p-value?» — спрашивают ВСЕГДА |
| Мощность | ⭐ **Мощность = 1 - β = P(отклонить H₀ | H₁ верна)** | 🟢 Ключевое | Размер выборки для A/B теста |
| Уровень значимости | Выбор α, связь с практикой | 🟢 Смотреть | Почему α = 0.05 |
| Множественные сравнения | Проблема множественных сравнений, поправка Бонферрони | 🟡 Полезно | При множественных A/B тестах |

**⭐ Главные инсайты:**
- Проверка гипотез = рассуждение от противного: предполагаем H₀, смотрим противоречат ли данные
- p-value < α → отклоняем H₀. Но p-value НЕ равно вероятности, что H₀ верна!
- Мощность зависит от: размера эффекта, размера выборки, уровня значимости, дисперсии
- α контролируем напрямую; β уменьшаем через увеличение выборки

**Связь с собесами:** Проверка гипотез — основа A/B тестирования. В продуктовых компаниях (Яндекс, Ozon, Т-Банк) вопросы про p-value и ошибки I/II рода задают КАЖДОМУ кандидату.

**Пререквизиты:** Лекции 2-3 (свойства оценок, доверительные интервалы).

### Лекция 7: Простая гипотеза и простая альтернатива. Гипотезы о характеристиках

**Приоритет:** 🟢 Высокая

Лемма Неймана-Пирсона (наиболее мощный критерий). Отношение правдоподобия. Гипотезы о среднем: z-тест, t-тест. Гипотезы о дисперсии: критерий хи-квадрат.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Нейман-Пирсон | ⭐ **Лемма Неймана-Пирсона: оптимальный критерий через отношение правдоподобия** | 🟡 Полезно | Теоретический фундамент |
| Отн. правдоподобия | Likelihood ratio test: Λ = L(θ₀)/L(θ₁) | 🟡 Полезно | Общий принцип построения критериев |
| z-тест | ⭐ **z-тест: H₀: μ = μ₀ при известной σ** | 🟢 Ключевое | Базовый тест, основа A/B |
| t-тест | ⭐ **t-тест Стьюдента: H₀: μ = μ₀ при неизвестной σ** | 🟢 Ключевое | «Когда z-тест, когда t-тест?» |
| Двухвыборочный | ⭐ **Двухвыборочный t-тест: сравнение средних двух групп** | 🟢 Ключевое | Основа A/B тестирования |
| Хи-квадрат | Критерий хи-квадрат для дисперсии | 🟡 Полезно | Редко спрашивают |

**⭐ Главные инсайты:**
- Лемма Неймана-Пирсона: наиболее мощный критерий основан на отношении правдоподобия
- z-тест: при большой выборке или известной σ → нормальное распределение статистики
- t-тест: при малой выборке и неизвестной σ → распределение Стьюдента с n-1 степенями свободы
- Двухвыборочный t-тест: (X̄₁ - X̄₂) / se → t-статистика для A/B теста

**Связь с собесами:** t-тест — рабочая лошадка A/B тестирования. «Какой тест использовать для сравнения средних?» — стандартный вопрос.

**Пререквизиты:** Лекция 6.

### Лекция 8: Гипотезы согласия. Гипотезы однородности. A/B тесты ⭐

**Приоритет:** 🟢 **КРИТИЧЕСКАЯ**

Критерии согласия (хи-квадрат Пирсона, Колмогорова-Смирнова). Гипотезы однородности. A/B тесты: дизайн, размер выборки, множественные сравнения.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Хи-квадрат Пирсона | ⭐ **Критерий хи-квадрат: Σ(O-E)²/E для проверки распределения** | 🟢 Смотреть | Классический критерий |
| Колмогоров-Смирнов | Критерий Колмогорова-Смирнова: sup|Fₙ(x) - F₀(x)| | 🟡 Полезно | Непараметрическая альтернатива |
| Однородность | ⭐ **Гипотезы однородности: одинаковы ли распределения двух выборок** | 🟢 Ключевое | Основа A/B тестов |
| A/B тесты | ⭐ **A/B тесты: дизайн, выбор метрики, размер выборки** | 🟢 Ключевое | Главная тема для продуктовых компаний |
| Размер выборки | ⭐ **Power analysis: n = f(α, β, effect size, σ²)** | 🟢 Ключевое | «Как рассчитать размер выборки для A/B теста?» |
| Множественные | Поправка на множественные сравнения (Бонферрони, FDR) | 🟢 Смотреть | При нескольких метриках/сегментах |
| Ранговые критерии | Критерий Манна-Уитни, ранговые тесты | 🟡 Полезно | Непараметрические альтернативы |

**⭐ Главные инсайты:**
- Гипотезы согласия: «соответствуют ли данные модели?» — хи-квадрат и KS-тест
- A/B тест = двухвыборочная проверка гипотез: контроль vs эксперимент
- Размер выборки: n ∝ (z_α + z_β)² · σ² / δ², где δ — минимальный детектируемый эффект
- Множественные сравнения: если проверяете k гипотез, скорректируйте α → α/k (Бонферрони)
- Ранговые критерии не предполагают нормальность — робастная альтернатива

**Связь с собесами:** A/B тестирование — **главная тема статистических вопросов** на MLE собеседованиях в продуктовых компаниях. «Как спроектировать A/B тест?», «Как рассчитать размер выборки?», «Что делать если метрика не нормальная?» — спрашивают ВСЕГДА.

**Пререквизиты:** Лекции 6-7.

### Лекция 9: Дисперсионный анализ. Корреляционный анализ. Линейная регрессия

**Приоритет:** 🟢 Высокая

ANOVA (дисперсионный анализ). Корреляция Пирсона и Спирмена. Ранговые методы. Введение в линейную регрессию.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Ранги | Ранги и ранговые статистики, их свойства | 🟡 Полезно | Основа для ранговых тестов |
| ANOVA | ⭐ **Дисперсионный анализ: разложение дисперсии на межгрупповую и внутригрупповую** | 🟢 Смотреть | «Как сравнить средние более чем 2 групп?» |
| F-статистика | F-критерий: отношение межгрупповой к внутригрупповой дисперсии | 🟢 Смотреть | Используется в регрессии для проверки значимости |
| Корреляция Пирсона | ⭐ **Корреляция Пирсона: r = cov(X,Y)/(σ_X · σ_Y)** | 🟢 Ключевое | «Чем корреляция отличается от причинности?» |
| Корреляция Спирмена | ⭐ **Корреляция Спирмена: ранговая корреляция** | 🟢 Смотреть | Робастная альтернатива Пирсону |
| Введение в регрессию | Линейная модель: Y = Xβ + ε | 🟢 Смотреть | Мостик к лекции 10 |

**⭐ Главные инсайты:**
- ANOVA = обобщение t-теста на >2 групп: F = MS_between / MS_within
- Корреляция Пирсона линейна, Спирмена — монотонна; корреляция ≠ причинность
- Ранговая корреляция робастна к выбросам и не предполагает нормальность
- ANOVA — фундамент для F-теста в регрессии (проверка значимости модели целиком)

**Связь с собесами:** «Корреляция vs причинность» — классический вопрос. ANOVA полезна для понимания F-теста в регрессии.

**Пререквизиты:** Лекции 6-8.

### Лекция 10: Линейная регрессия ⭐

**Приоритет:** 🟢 **КРИТИЧЕСКАЯ**

МНК (OLS): матричная формулировка, теорема Гаусса-Маркова. Проверка гипотез о коэффициентах. R² и скорректированный R². Диагностика модели. Мультиколлинеарность.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Матричная форма | ⭐ **Y = Xβ + ε, β̂ = (X^TX)⁻¹X^Ty** | 🟢 Ключевое | «Как аналитически решить линейную регрессию?» |
| Гаусс-Марков | ⭐ **Теорема Гаусса-Маркова: OLS = BLUE (best linear unbiased estimator)** | 🟢 Ключевое | OLS оптимален среди линейных несмещённых |
| Свойства β̂ | E[β̂] = β, Var(β̂) = σ²(X^TX)⁻¹ | 🟢 Смотреть | Распределение оценок коэффициентов |
| Значимость коэфф. | ⭐ **t-тест для отдельных коэффициентов: t = β̂ⱼ/se(β̂ⱼ)** | 🟢 Ключевое | «Значим ли этот признак?» |
| F-тест модели | ⭐ **F-тест: значимость модели в целом** | 🟢 Смотреть | Связь с ANOVA |
| R² | ⭐ **R² = 1 - SS_res/SS_tot, скорректированный R²** | 🟢 Ключевое | «Что означает R²? Почему нужен скорректированный?» |
| Диагностика | Анализ остатков, гомоскедастичность, нормальность | 🟢 Смотреть | Проверка предположений модели |
| Мультиколлинеарность | ⭐ **Мультиколлинеарность: VIF, проблемы с (X^TX)⁻¹** | 🟢 Ключевое | «Зачем нужна регуляризация?» → Ridge/Lasso |

**⭐ Главные инсайты:**
- OLS минимизирует ||Y - Xβ||² → нормальное уравнение β̂ = (X^TX)⁻¹X^Ty
- Гаусс-Марков: при E[ε] = 0, Var(ε) = σ²I, OLS — лучшая среди линейных несмещённых оценок
- R² растёт при добавлении признаков → скорректированный R² штрафует за сложность
- Мультиколлинеарность раздувает Var(β̂) → регуляризация (Ridge) добавляет λI к X^TX
- Анализ остатков: паттерны = нарушение предположений модели

**Связь с собесами:** Линейная регрессия — один из «5 обязательных выводов» на MLE собеседованиях. Вопросы: OLS вывод, R², мультиколлинеарность, связь с MLE, когда OLS плохо работает.

**Пререквизиты:** Лекции 4 (MLE), 9 (ANOVA, корреляция).

### Лекция 12: Байесовский подход ⭐

**Приоритет:** 🟢 Высокая

Байесовская статистика: априорное распределение, апостериорное распределение, формула Байеса. MAP-оценка. Сопряжённые распределения. Байесовская линейная регрессия. Связь с регуляризацией.

| Блок | Содержание | Статус | Для собесов |
|------|-----------|--------|-------------|
| Философия | Частотный vs байесовский подход к вероятности | 🟡 Полезно | Контекст |
| Формула Байеса | ⭐ **P(θ|data) ∝ P(data|θ) · P(θ)** | 🟢 Ключевое | Фундамент байесовского подхода |
| Априорное | ⭐ **Априорное распределение: выбор, информативные vs неинформативные** | 🟢 Смотреть | «Что такое prior?» |
| Апостериорное | ⭐ **Апостериорное распределение: обновление убеждений через данные** | 🟢 Ключевое | Posterior ∝ Likelihood × Prior |
| MAP | ⭐ **MAP-оценка: argmax P(θ|data) = argmax [logL(θ) + logP(θ)]** | 🟢 Ключевое | MAP с нормальным prior = Ridge |
| Сопряжённые | Сопряжённые распределения: Beta-Bernoulli, Normal-Normal | 🟡 Полезно | Красивая математика |
| Байес. регрессия | ⭐ **Байесовская линейная регрессия: posterior на коэффициенты** | 🟢 Смотреть | Связь с Ridge regression |
| Предсказательное | Предсказательное распределение: P(y_new|data) | 🟡 Полезно | Неопределённость предсказаний |
| Регуляризация | ⭐ **MAP + Gaussian prior = Ridge; MAP + Laplace prior = Lasso** | 🟢 Ключевое | Байесовская интерпретация регуляризации |

**⭐ Главные инсайты:**
- Байесовский подход: параметр — случайная величина с распределением, а не фиксированная константа
- MAP = MLE + регуляризация: log-prior играет роль штрафа
- Gaussian prior → L2 (Ridge), Laplace prior → L1 (Lasso) — элегантная связь
- Байесовский подход естественно даёт неопределённость предсказаний (а не только точечную оценку)
- С ростом данных posterior «сжимается» — данные доминируют над prior

**Связь с собесами:** «Какова байесовская интерпретация регуляризации?» — вопрос для Senior/Staff позиций. MAP vs MLE — стандартный вопрос на продвинутых собеседованиях.

**Пререквизиты:** Лекции 4 (MLE), 10 (линейная регрессия).
