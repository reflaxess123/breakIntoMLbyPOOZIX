### §1. Вероятностная модель эксперимента с конечным числом исходов (стр. 21–35)

**Статус:** ✅ РАЗОБРАН

**Что покрыто:**
- Пространство элементарных исходов Ω
- Событие как подмножество Ω
- Алгебра событий: замкнутость относительно ∪, ∩, \
- σ-алгебра: усиление — замкнутость относительно счётных объединений
- Вероятностная мера P: три аксиомы Колмогорова (неотрицательность, нормировка, счётная аддитивность)
- Вероятностное пространство (Ω, 𝒜, P)
- Разбиение и атомы
- Классическая вероятность P(A) = N(A)/N
- Комбинаторика: перестановки, размещения, сочетания, выборки с возвращением
- Задача о совпадениях (birthday problem): точная формула и приближение через e
- Зачем нужна σ-алгебра: обычная алгебра не работает с бесконечными объединениями
- Парадокс Банаха-Тарского: почему нельзя определить меру на всех подмножествах
- Борелевская σ-алгебра: счётные кирпичики → непрерывные конструкции

**Статус:** ✅ РАЗОБРАН

### §3. Условные вероятности. Независимость (стр. 43–52)

**Статус:** ❌ НЕ РАЗОБРАН

**Приоритет:** 🟢 КРИТИЧЕСКИЙ — нельзя пропустить

**Что тут:**
- Определение условной вероятности P(A|B) = P(A∩B) / P(B)
- Формула умножения: P(A∩B) = P(A|B) · P(B)
- Формула полной вероятности: P(A) = Σ P(A|Bᵢ) · P(Bᵢ)
- Формула Байеса (формально, через полную вероятность)
- Независимость событий: P(A∩B) = P(A) · P(B)
- Попарная vs взаимная независимость
- Цепное правило: P(A₁∩A₂∩...∩Aₙ) = P(A₁) · P(A₂|A₁) · P(A₃|A₁∩A₂) · ...

**Зачем для ML:**
- Формула Байеса — фундамент Naive Bayes, байесовского вывода, posterior estimation
- Независимость — ключевое предположение в MLE (iid), биномиальном распределении
- Цепное правило — авторегрессивные модели (GPT): P(x₁,...,xₙ) = P(x₁)·P(x₂|x₁)·...
- Условная вероятность → условное распределение → условное ожидание

**Что ты уже знаешь:**
- Байес через prior × likelihood = posterior (интуитивно, через демо)
- Но формально через полную вероятность и формулу умножения — нет

**Можно ли пропустить:** НЕТ. §4 строится на условных вероятностях.

### §4. Случайные величины и их характеристики (стр. 53–66)

**Статус:** ❌ НЕ РАЗОБРАН

**Приоритет:** 🟢 КРИТИЧЕСКИЙ — ядро всей теории

**Что тут:**
- Определение случайной величины: функция X: Ω → ℝ
- Функция распределения F(x) = P(X < x)
- Дискретные и непрерывные случайные величины
- Плотность вероятности f(x) = F'(x)
- Математическое ожидание E[X] — определение, свойства, линейность
- Дисперсия Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²
- Моменты высших порядков
- Ковариация, корреляция (формально)
- Неравенство Чебышёва: P(|X - E[X]| ≥ ε) ≤ Var(X)/ε²
- Производящие функции

**Зачем для ML:**
- Случайная величина — это то, с чем ты работаешь всегда (потери, веса, данные)
- E[X] — матожидание = что оптимизируем (empirical risk minimization)
- Var(X) — разброс = нестабильность модели
- F(x), f(x) — функция распределения и плотность = основа для MLE
- Неравенство Чебышёва — теоретическая гарантия для ЗБЧ

**Что ты уже знаешь:**
- E[X], Var(X) — формулы и интуицию
- CDF — эмпирическую
- Ковариацию, корреляцию — пройдено

**Что будет новым:**
- Формальное определение случайной величины (через измеримость)
- Плотность как производная CDF
- Неравенство Чебышёва
- Производящие функции

**Можно ли пропустить:** НЕТ. Без случайных величин §5 и §6 бессмысленны.

### §5. Схема Бернулли. I. Закон больших чисел (стр. 67–77)

**Статус:** ❌ НЕ РАЗОБРАН

**Приоритет:** 🟢 ВЫСОКИЙ

**Что тут:**
- Схема Бернулли: n независимых испытаний с двумя исходами
- Частота успехов Sₙ/n
- Закон больших чисел (ЗБЧ): Sₙ/n → p при n → ∞
- Доказательство через неравенство Чебышёва
- Связь частоты с вероятностью

**Зачем для ML:**
- ЗБЧ — почему обучение на данных работает: эмпирическое среднее → истинное среднее
- Empirical Risk Minimization: 1/n Σ L(yᵢ, ŷᵢ) → E[L(y, ŷ)]
- Сходимость SGD: почему mini-batch gradient приближает true gradient

**Что ты уже знаешь:**
- ЗБЧ концептуально — среднее стабилизируется с ростом выборки

**Что будет новым:**
- Формальное доказательство через Чебышёва
- Связь с схемой Бернулли

**Можно ли читать выборочно:** ДА. Основные теоремы и их смысл, доказательства можно пробежать быстро.

### §6. Схема Бернулли. II. Предельные теоремы (стр. 78–93)

**Статус:** ❌ НЕ РАЗОБРАН

**Приоритет:** 🟢 ВЫСОКИЙ

**Что тут:**
- Локальная теорема Муавра-Лапласа: биномиальное → нормальное (для отдельных значений)
- Интегральная теорема Муавра-Лапласа: биномиальное → нормальное (для интервалов)
- Теорема Пуассона: биномиальное → пуассоновское (при малых p)
- Приближение биномиальных вероятностей

**Зачем для ML:**
- Муавра-Лаплас — это ЦПТ для частного случая. Объясняет откуда берётся нормальное распределение
- Пуассон — редкие события (аномалии, клики, отказы)
- Понимание почему нормальное распределение так часто встречается

**Что ты уже знаешь:**
- ЦПТ — средние выборок → нормальное распределение
- Биномиальное распределение и его графики (видел что при больших n → колокол)

**Что будет новым:**
- Формальные формулировки предельных теорем
- Пуассоново приближение

**Можно ли читать выборочно:** ДА. Главное — интегральная теорема Муавра-Лапласа (= ЦПТ для схемы Бернулли). Остальное — для эрудиции.
